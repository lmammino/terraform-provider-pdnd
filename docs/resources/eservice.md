# pdnd_eservice (Resource)

Manages a PDND E-Service, including creation with an initial descriptor.

The e-service is created with an initial descriptor in DRAFT state. To manage additional descriptors or publish the initial descriptor, use the `pdnd_eservice_descriptor` resource.

## Example Usage

```terraform
# Create an e-service with an initial descriptor
resource "pdnd_eservice" "example" {
  name        = "My API Service"
  description = "A REST API service for data exchange"
  technology  = "REST"
  mode        = "DELIVER"

  initial_descriptor_agreement_approval_policy = "AUTOMATIC"
  initial_descriptor_audience                  = ["api.example.com"]
  initial_descriptor_daily_calls_per_consumer  = 1000
  initial_descriptor_daily_calls_total         = 10000
  initial_descriptor_voucher_lifespan          = 3600
}
```

## Schema

### Required

- `description` (String) E-Service description (10-250 characters)
- `initial_descriptor_agreement_approval_policy` (String) Agreement approval policy for the initial descriptor: AUTOMATIC or MANUAL
- `initial_descriptor_audience` (List of String) Audience for the initial descriptor
- `initial_descriptor_daily_calls_per_consumer` (Number) Daily calls per consumer for the initial descriptor (1-1,000,000,000)
- `initial_descriptor_daily_calls_total` (Number) Total daily calls for the initial descriptor (1-1,000,000,000)
- `initial_descriptor_voucher_lifespan` (Number) Voucher lifespan in seconds for the initial descriptor (60-86,400)
- `mode` (String) E-Service mode: RECEIVE or DELIVER
- `name` (String) E-Service name (5-60 characters)
- `technology` (String) E-Service technology: REST or SOAP

### Optional

- `initial_descriptor_description` (String) Description for the initial descriptor
- `is_client_access_delegable` (Boolean) Whether client access can be delegated
- `is_consumer_delegable` (Boolean) Whether consumers can delegate
- `is_signal_hub_enabled` (Boolean) Whether the signal hub is enabled
- `personal_data` (Boolean) Whether the e-service handles personal data

### Read-Only

- `id` (String) E-Service UUID
- `initial_descriptor_id` (String) Initial descriptor UUID
- `producer_id` (String) Producer UUID
- `template_id` (String) Template UUID
